## 1. 产品概述
构建一个INFP风格、Dark Academia美学并融合Retro-System（复古终端）美学的个人网站模板。
- 这是一个“数字舱口”（Digital Hatch）。
- 完全抛弃原生滚动条。页面静止，但在原地发生惊艳的视觉演算。
- 核心是一个从混沌打结状态，随鼠标滚轮徐徐铺开的知识图谱（Mind Map）。

## 2. 核心功能

### 2.1 节点定义与数据标识
抛弃虚假内容。节点带有“诗意名称”与底层系统的十六进制逻辑地址：
- **大节点**：
  - INFP (ADDR_0x0A1)
  - AI Native (ADDR_0x0A2)
  - 10後 (ADDR_0x0A3)
  - ADHD (ADDR_0x0A4)
  - 天生反骨 (ADDR_0x0A5)
  - 貓奴 (ADDR_0x0A6)
  - 宅 (ADDR_0x0A7)
  - OI (ADDR_0x0A8)
  - 數競 (ADDR_0x0A9)
- **小节点**：
  - 音樂 (ADDR_0x0B1)
  - 文學 (ADDR_0x0B2)
  - 氣象 (ADDR_0x0B3)

### 2.2 功能模块
1. **终端启动序列 (Terminal Boot-up)**:
   - 全黑背景的 Cognitive Kernel 加载界面。
   - 模拟旧式系统的自检与启动流，随后切入图谱。
2. **滚轮驱动的图谱解构 (Scroll-driven Graph Unfolding)**:
   - 图谱初始状态是“打结”的混沌团块。
   - 监听滚轮事件（代替页面滚动），驱动物理引擎的斥力/引力系数。
   - 滚轮向下，图谱徐徐铺开，突触延展；滚轮向上，图谱重新收缩打结。
3. **CRT 模拟器美学 (Retro-System Aesthetic)**:
   - **扫描线 (Scanlines)**：覆盖全屏的动态移动扫描线动画。
   - **暗角与闪烁 (CRT FX)**：四周深色 Vignette 描边，伴随极轻微的屏幕电流闪烁感 (Flicker)，营造“深夜操作旧终端”的氛围。
4. **零重力交互 (Non-Scrolling Terminal)**:
   - 彻底禁用向下滚动（`overflow: hidden`）。
   - 节点悬浮、拖拽带有物理反馈，伴随环境视差。

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页 | 终端启动层 | 模拟 Cognitive Kernel 加载，短暂的字符跳动后揭幕 |
| 主页 | 知识图谱中枢 | 初始打结，随滚轮铺开，带有 Hex 地址的节点 |
| 主页 | CRT 视觉滤镜 | 扫描线、暗角、轻微电流闪烁的全局覆盖层 |

## 3. 核心流程
用户访问 -> 看到黑屏 Cognitive Kernel 终端启动 -> 启动完毕，显示一团纠缠打结的节点 -> 用户本能滑动滚轮 -> 图谱在原地像神经元一样徐徐展开 -> 节点下方浮现十六进制地址 -> 鼠标移动产生视差，悬浮/拖拽节点产生物理阻尼。

## 4. 用户界面设计
### 4.1 设计风格
- **主色调**：深邃的终端黑（#000000 - #0A0A0A），复古荧光绿/琥珀色点缀（#00FF41 或 #FFB000），或者暗调黄铜金以契合 Dark Academia。
- **字体**：
  - 节点诗意名称：Playfair Display（古典、文学感）。
  - 十六进制地址/启动序列：VT323 或 Fira Code（终端、代码感）。
- **UI元素**：无卡片。纯粹的光晕、连线、噪点、扫描线和文字。
